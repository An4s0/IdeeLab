# Web App Routes Overview

This document outlines the routes in the web application and the purpose of each page.

## Routes

| Route              | Component            | Description                                                           |
| ------------------ | -------------------- | --------------------------------------------------------------------- |
| `/`                | `HomePage`           | The main landing page of the application.                             |
| `/terms`           | `TermsPage`          | The terms and conditions page.                                        |
| `/privacy`         | `PrivacyPage`        | The privacy policy page.                                              |
| `/login`           | `LoginPage`          | Allows existing users to log in to their accounts.                    |
| `/register`        | `RegisterPage`       | Page for new users to create an account.                              |
| `/forgot-password` | `ForgotPasswordPage` | Allows users to reset their password.                                 |
| `/reset-password`  | `ResetPasswordPage`  | Allows users to set a new password after requesting a reset.          |
| `/logout`          | `LogoutPage`         | Logs the user out of their account.                                   |
| `/verify-email`    | `VerifyEmailPage`    | Page for users to verify their email address after registration.      |
| `/challenge`       | `ChallengePage`      | Shows a list of all challenges with categories and difficulty levels. |
